# Screenshot Selection

The original upload contained many screenshots from different projects. Only screenshots directly related to the Wazuh SIEM Security Lab were selected.

## Included Screenshots

- `aws-vpc-workflow.png`
- `ec2-instances-overview.png`
- `wazuh-deployment-dashboard.png`
- `wazuh-agent-connected.png`
- `wazuh-vulnerability-dashboard.png`
- `suricata-threat-hunting-alerts.png`
- `virustotal-file-integrity-alerts.png`
- `suricata-service-running.png`

## Selection Criteria

A screenshot was selected only if it showed one of the following:

- AWS cloud networking or EC2 setup for the lab
- Wazuh deployment or dashboard
- Agent connection
- Vulnerability detection
- Suricata IDS monitoring
- VirusTotal integration
- File integrity monitoring
- Security alert investigation
